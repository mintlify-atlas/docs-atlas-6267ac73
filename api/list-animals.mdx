---
title: 'List animals'
description: 'Returns all available animal endpoints with their titles and descriptions'
---

`GET /api/animals`

Returns an array of all registered animal endpoints. Use this to discover which animals are available and build dynamic UIs.

## Parameters

This endpoint takes no parameters.

## Response

<ResponseField name="[]">array of animal info objects</ResponseField>

<ResponseField name="endpoint" type="string" required>
  The URL slug used to access the animal. Use this as the path parameter for the image and count endpoints.
</ResponseField>

<ResponseField name="title" type="string" required>
  Human-readable display name for the animal.
</ResponseField>

<ResponseField name="description" type="string" required>
  Short description of the animal.
</ResponseField>

## Example

```bash
curl http://localhost:8080/api/animals
```
