## Prerequisites

- [Go 1.24+](https://go.dev/dl/)
- A running [PocketBase](https://pocketbase.io) instance (optional — the `/floppapi` endpoint works without one)

## Run the API locally

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/Padi142/floppa-api.git
    cd floppa-api
    ```
  </Step>

  <Step title="Create config.json">
    Copy the example config and fill in your PocketBase credentials:

    ```bash
    cp backend/config.json.example backend/config.json
    ```

    Edit `backend/config.json`:

    ```json backend/config.json
    {
      "pocketbase_url": "http://localhost:8090",
      "pocketbase_admin": "admin@example.com",
      "pocketbase_password": "your_password"
    }
    ```

    <Note>
      The `/floppapi` endpoint serves images from the local `./floppa` directory and does not require PocketBase. If you only want to test that endpoint, you can point `pocketbase_url` at any URL — it won't be called.
    </Note>
  </Step>

  <Step title="Start the server">
    ```bash
    cd backend
    go run .
    ```

    You should see:

    ```
    [GIN-debug] Listening and serving HTTP on :8080
    ```
  </Step>

  <Step title="Fetch your first image">
    The `/floppapi` endpoint serves local images and works immediately:

    ```bash
    curl http://localhost:8080/floppapi --output floppa.jpg
    ```

    To test a PocketBase-backed animal:

    ```bash
    curl http://localhost:8080/capybara --output capybara.jpg
    ```
  </Step>

  <Step title="Open the frontend (optional)">
    Navigate to [http://localhost:8080](http://localhost:8080) in your browser to see the Svelte frontend. The admin panel is available at [http://localhost:8080/admin](http://localhost:8080/admin).
  </Step>
</Steps>
